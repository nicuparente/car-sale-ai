# Car-Sale-AI
Vets in tech AI/ML Course project to deploy an AI model trained on car sales which is accessible by a web api


## Model Info

**Data Information**
 
 The data used to train the model was from a Kaggle training data set https://www.kaggle.com/datasets/missionjee/car-sales-report. This data set seems to be synthetically created.
